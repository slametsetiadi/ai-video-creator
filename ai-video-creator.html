📖 Panduan Instalasi & Penggunaan

📦 Langkah 1: Instalasi (Sangat Mudah!)

  1. Simpan file ini dengan nama ai-video-creator.html
  2. Buka file tersebut di browser (Chrome/Edge/Firefox/Safari)
  3. Selesai! Aplikasi langsung berjalan tanpa instalasi server

✅ Tidak perlu install Node.js, Python, atau server apapun!

🔑 Langkah 2: Setup API Keys

  1. Pexels API: Daftar di pexels.com → Profile → API → Copy API Key
  2. Pixabay API: Daftar di pixabay.com → Dashboard → API Key
  3. Groq AI (Opsional): Daftar di console.groq.com → API Keys → Create
  4. Gemini AI (Opsional): Daftar di aistudio.google.com → Get API Key
  5. Masukkan API Key di tab "Pengaturan API" → Klik "Test" untuk verifikasi
  6. Klik "Simpan Semua Pengaturan" untuk menyimpan

🎬 Langkah 3: Cara Menggunakan

  1. Cari Video: Buka tab "Cari Video" → Tekan tombol mikrofon dan bicarakan tema, atau ketik keyword → Klik "Cari Video"
  2. Pilih Video: Klik video yang ingin digunakan (bisa pilih beberapa)
  3. Editor: Buka tab "Editor" → Upload audio voiceover → Pilih transisi & resolusi → Aktifkan fitur auto
  4. Export: Buka tab "Export" → Klik "Mulai Render" → Download video → Generate SEO

💡 Tips & Catatan Penting

  • Semua video dari Pexels & Pixabay adalah GRATIS dan BEBAS COPYRIGHT
  • API Key disimpan di browser (localStorage) - aman dan privat
  • Fitur Voice Recognition bekerja terbaik di Chrome/Edge
  • Untuk hasil terbaik, gunakan audio voiceover yang jelas
  • Render video dilakukan di browser - pastikan RAM cukup (min 4GB)
  • AI Provider (Groq/Gemini) digunakan untuk generate SEO content yang lebih baik
  • Semua processing dilakukan secara lokal - tidak ada data yang dikirim ke server kecuali API calls
